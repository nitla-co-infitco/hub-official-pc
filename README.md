# hub-official-pc

本项目是一个用于归档和发布多个独立 HTML 页面的仓库。这些页面内容相互独立，不针对任何特定域名或具体网站，仅作为静态资源存储与展示使用。

## 项目简介

本仓库旨在集中管理多个独立的 HTML 页面，便于版本控制与历史追溯。所有页面均为静态内容，不依赖后端服务，可直接通过浏览器打开查看。仓库本身不提供任何形式的网络托管或动态服务。

## 目录说明

```
hub-official-pc/
├── pages/          # 存放所有独立 HTML 页面文件
├── assets/         # 页面引用的公共资源（如图片、样式、脚本）
└── README.md       # 本文件
```

- **pages/**：每个页面以独立文件夹或文件形式存放，命名清晰，便于识别。
- **assets/**：存放页面间共用的静态资源，避免重复引用。

## 页面归档说明

- 每个 HTML 页面均为自包含的独立文件，可单独运行。
- 页面内容不涉及具体网站、域名或商业推广信息。
- 归档目的仅为保存与展示，不保证内容长期有效或更新。
- 页面可能引用外部资源（如字体、图标库），请确保网络连通性。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时需遵循已有目录结构。
- 欢迎提交 Issue 或 Pull Request，但请注意保持内容中立、不涉及具体域名或网址。
- 对于不符合归档原则的页面，维护者有权拒绝合并或直接移除。
- 本仓库不承担任何因页面内容引发的法律责任。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在 `pages/` 目录下找到对应页面文件。
3. 使用浏览器直接打开 `.html` 文件即可查看。
